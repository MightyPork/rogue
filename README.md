Rogue - dungeon crawler
=======================

Goals
-----

- Simple retro-themed dungeon crawler
- Multiplayer support


Features
--------

- Full OOP design
- Event driven
- OpenGL 2D rendering

- Random floors
- Real-time gameplay
- Monsters with AI (-> combat system)


Possibly added
--------------

- Stats and leveling
- Collectable items
- Potions, food
- Simple inventory system


Used libraries
--------------

- Slick2D
- LWJGL
