Dungeon crawler RPG
===================

Goals
-----

- Simple retro-themed dungeon crawler
- Multiplayer support


Features
--------

- Randomly generated floors
- Hybrid turn-based gameplay
- Stats and leveling
- Monsters
- Collectable items
- Potions, food
- Simple inventory system


Used libraries
--------------

- Slick2D
- NiftyGUI
- LWJGL
